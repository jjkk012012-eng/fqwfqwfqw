STEP 견적 자동 산출 V50 Clean Design

반영 내용
- 좌측 상단 로고에 V50 Clean Design 버전 표시
- 공법 추천을 단순 if문이 아니라 판금/사출/CNC 점수형 결정 로직으로 재작성
- 구매품/프로파일/선반/3D프린팅은 이름과 형상 힌트로 먼저 제외
- 판금/절곡
  - 1~20mm 동일두께 추정
  - 단순 판/접힌 판/종이접기형
  - 얇고 넓은 비율, 낮은 체적률, 평면 조합 형상
  - 리브/보스/스냅이 없을 때 판금 우선
- 사출
  - 리브, 보스, 스냅, 클립, 훅, 속 빈 케이스형, 수지/금형 힌트가 있을 때만 강하게 추천
- CNC/MCT
  - 리브 없는 일반 기하학 형상
  - 블록, 지그, 마운트, 포켓/슬롯/가공형 구조
  - 10mm 이내 복잡 형상이어도 리브/보스가 없으면 CNC 우선
- 선택 파트 상세에 판금/사출/CNC 점수 표시
- 이전 브라우저 학습값이 새 로직을 덮어쓰지 않도록 V50 Clean Design 학습키로 분리

GitHub Pages 업로드
ZIP을 풀고, 폴더 안의 내용물(index.html, styles.css, app.js, README.txt, data, vendor)을 저장소 root에 덮어쓴 뒤 Commit/Push 하세요.

주의
STEP/STP에는 공법 정보가 직접 들어있지 않은 경우가 많아, 이 도구는 형상 기반 자동 추천 + 공장장 수정 확정 구조입니다.


[V50 Clean Design 변경]
- 판재는 규격 두께표 기준으로만 판금 후보 처리합니다.
- 1.8T, 1.3T, 2.7T처럼 표준 판재 규격과 맞지 않는 추정두께는 판금에서 제외하고 CNC/MCT 쪽으로 보냅니다.
- 판금/커버/패널/브라켓 계열 파트명 + 표준두께 + 단순 형상은 판금/절곡으로 우선 처리합니다.
- 리브/보스/스냅/클립은 사출, 리브 없는 복잡/일반 기하학 형상은 CNC/MCT로 처리합니다.
