STEP 견적 자동 산출 V35

- 공장용 STEP/STP 견적 도구
- 어셈블리/서브어셈블리는 표에서 제외하고 말단 파트만 표시
- 선택 파트만 3D 뷰어에 크게 표시
- 공법 추천 로직 재작성: 구매품 > 프로파일 > 선반 > 판금/절곡 > CNC/MCT > 사출/3D프린팅
- 판금/절곡: 같은 두께의 판재형이면 절곡 여부와 관계없이 판금/절곡 추천
- CNC/MCT: 구매품/프로파일/선반/판금을 제외한 두꺼운 절삭 가공품 추천
- 공법을 사용자가 수정하면 같은 파트명/유사 파트명에 대해 브라우저에 학습 저장
- 단가표 저장/다운로드/불러오기/붙여넣기 지원
