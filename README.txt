STEP 견적 자동 산출 V21 - 어셈블리 비표시 버전

변경 사항
- 어셈블리/전체 조립품은 화면에 표시하지 않습니다.
- STEP 업로드 후 첫 번째 말단 파트만 뷰어에 표시합니다.
- 파트 행을 클릭하면 해당 파트만 isolate 표시합니다.
- 구매품도 mesh가 있으면 선택 시 확대 표시합니다.
- mesh 매칭이 없는 파트는 전체 어셈블리 대신 경고만 표시하고, 견적표 입력은 가능합니다.
- 표에는 말단 파트만 표시합니다.

업로드 방법
index.html, styles.css, app.js, README.txt, data 폴더, vendor 폴더를 GitHub 저장소 root에 덮어쓰기하세요.
