STEP 견적 자동 산출 V18

- 어셈블리/서브어셈블리는 표에서 제외하고 파트만 표시합니다.
- 구매품: 수량 × 구매단가 × 마진. 구매단가는 표에서 수정합니다.
- 판금/절곡: kg/개 × 판재 kg단가 + 절곡수 × 절곡단가. 절곡수는 사람이 입력합니다.
- CNC/MCT, 선반, 사출, 3D프린팅: kg/개 × 재질별 kg단가 + 시간/개 × 공통 시간당 단가 + 셋업비. 시간/개는 사람이 입력합니다.
- 압출/프로파일: kg/개 × 압출 kg단가 + 가공비/개.
- 용접: 자동 견적 제외, 내부 검토 표시.
- 단가표는 다운로드/불러오기/복사/붙여넣기/브라우저 저장이 가능합니다.
