STEP 견적 자동 산출 V34 - 서비스 런칭 구조 재작성본

업로드 방법
1. 이 폴더 안의 내용물(index.html, styles.css, app.js, data, vendor)을 GitHub 저장소 root에 넣습니다.
2. GitHub Desktop에서 Commit 후 Push 합니다.
3. GitHub Pages 주소에서 Ctrl+F5로 새로고침합니다.

핵심 구조
- 어셈블리/서브어셈블리는 표에서 제거하고 말단 파트만 표시합니다.
- 3D 뷰어는 선택한 파트만 크게 표시합니다.
- 구매품은 수량과 구매단가만 수정합니다.
- 판금/절곡은 절곡수만 입력합니다.
- CNC/MCT, 선반, 사출, 3D프린팅은 시간/개를 입력합니다.
- 압출은 가공비/개를 입력합니다.
- 재질별 kg 단가, 공정별 시간당 단가, 공법별 마진은 단가표에서 수정합니다.
- 단가표는 브라우저 저장, 다운로드, 불러오기, 복사/붙여넣기를 지원합니다.

용량
- 압축 해제 후 약 90MB대입니다.
- 단일 파일은 GitHub 100MB 제한 이하입니다.
- data/recommendation_reference_90mb.json은 향후 추천 고도화용 참고 데이터이며, 앱 실행에는 직접 로드하지 않습니다.
