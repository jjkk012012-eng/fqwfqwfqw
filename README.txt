STEP 견적 자동 산출 V48

반영 내용
- 좌측 상단 로고에 V48 버전 표시
- 공법 추천 로직 재정리
  1) 구매품/프로파일/선반 우선 제외
  2) 리브·보스·스냅·클립·훅은 사출 우선
  3) 1~20mm 동일두께 + 단순 판/접힌 판/종이접기형은 판금/절곡으로 CNC보다 먼저 확정
  4) 10mm 이내 + 복잡 형상은 리브 있으면 사출, 리브 없으면 CNC/MCT
  5) 일반 기하학 솔리드는 CNC/MCT
- 판금 판정 개선
  - bbox minDim만 보지 않고 2V/A 추정두께, 낮은 체적률, 평면 방향 비율, 접힌 판 구조를 같이 봅니다.
  - 선택 파트 상세에 판금후보/접힌판/평면비율/리브보스 지표 표시
- 이전 V47 브라우저 학습값이 새 로직을 덮어쓰지 않도록 V48 학습키로 분리
