공장용 STEP 견적 계산기 - 안정 파서 버전

실행:
1) index.html 더블클릭 또는 GitHub Pages 업로드
2) STEP/STP 파일 선택
3) 말단 파트표에서 공법/재질/수량/탭/절곡/마진 수정

이번 버전은 외부 CDN/OCCT 의존 없이 STEP 텍스트 구조를 직접 읽습니다.
어셈블리/서브어셈블리 컨테이너는 견적 제외하고 말단 파트만 표시합니다.

주의:
- 실제 3D 형상 렌더링은 뺀 안정 파서 버전입니다.
- 절곡은 실제 곡면을 3D로 판정하지 않고 이름/두께 힌트 기반 후보만 넣습니다.
- 불확실하면 절곡 0회로 두고 공장이 직접 수정하도록 설계했습니다.
