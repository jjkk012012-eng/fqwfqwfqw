STEP 견적 자동 산출 V45 - 동일두께 사출/판금 분리 로직

업로드 방법
1. 이 ZIP을 압축 해제합니다.
2. 폴더 안의 index.html, app.js, styles.css, README.txt, data, vendor를 GitHub Pages 저장소 루트에 덮어씁니다.
3. Commit / Push 후 브라우저에서 Ctrl+F5로 새로고침합니다.

V45 핵심 로직
- 구매품/프로파일/선반을 먼저 제외합니다.
- 사출도 동일 두께일 수 있으므로, 동일 두께만으로 판금 처리하지 않습니다.
- 리브, 보스, 스냅, 클립, 훅, 얇은 벽의 복잡 피처가 있으면 사출로 추천합니다.
- 종이접기처럼 넓은 판재가 접힌 형상은 판금/절곡으로 추천합니다.
- 리브 없는 일반 기하학적 솔리드 형상은 CNC/MCT로 추천합니다.
- 공장장이 수정한 공법은 브라우저에 학습되어 다음 업로드 때 반영됩니다.

주의
- data/recommendation_reference_90mb.json은 GitHub 용량 기준을 맞춘 참고 데이터이며 앱 실행에는 직접 로드되지 않습니다.
- 단일 파일은 GitHub 100MB 제한 이하입니다.


V47 변경사항
- 공법 로직: 1~20mm 동일 두께 + 단순 판형은 판금/절곡, 10mm 이내 복잡 형상은 리브/보스 있으면 사출, 리브 없는 일반 기하학 형상은 CNC/MCT로 판단합니다.
- 견적표 헤더(파트/수량/추천/공법/재질/견적가 등)를 클릭하면 같은 조건끼리 묶어서 정렬합니다.


V47 공법 로직 재작성
- 구매품/프로파일/선반을 먼저 제외합니다.
- 리브/보스/스냅/클립/훅이 있으면 사출 우선입니다.
- 10mm 이내 + 복잡 형상은 판금으로 보내지 않고, 수지/사출 힌트가 있으면 사출, 아니면 CNC/MCT로 보냅니다.
- 1~20mm 동일두께 추정 + 단순 판/접힌 판 구조는 CNC보다 먼저 판금/절곡으로 확정합니다.
- 일반 기하학 솔리드는 CNC/MCT로 보냅니다.
