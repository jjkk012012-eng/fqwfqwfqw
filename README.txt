핵심 변경
- 어셈블리명은 상단에 고정 표시합니다.
- 표에는 어셈블리/서브어셈블리를 제외한 말단 파트만 표시합니다.
- 절곡은 자동으로 회수를 맞추려 하지 않고, 절곡품 여부를 먼저 판단합니다.
- 절곡수/홀수/구매단가/kg/개는 공장장이 표에서 바로 수정합니다.
- 봉/파이프/핸들/밸브/니플/리드류는 판금이 아니라 구매품 우선입니다.
- 판금은 이름+얇은 판재형 형상 조건이 맞을 때만 추천합니다.
