STEP 견적 자동 산출 V49

반영 내용
- 좌측 상단 로고에 V49 버전 표시
- 공법 추천을 단순 if문이 아니라 판금/사출/CNC 점수형 결정 로직으로 재작성
- 구매품/프로파일/선반/3D프린팅은 이름과 형상 힌트로 먼저 제외
- 판금/절곡
  - 1~20mm 동일두께 추정
  - 단순 판/접힌 판/종이접기형
  - 얇고 넓은 비율, 낮은 체적률, 평면 조합 형상
  - 리브/보스/스냅이 없을 때 판금 우선
- 사출
  - 리브, 보스, 스냅, 클립, 훅, 속 빈 케이스형, 수지/금형 힌트가 있을 때만 강하게 추천
- CNC/MCT
  - 리브 없는 일반 기하학 형상
  - 블록, 지그, 마운트, 포켓/슬롯/가공형 구조
  - 10mm 이내 복잡 형상이어도 리브/보스가 없으면 CNC 우선
- 선택 파트 상세에 판금/사출/CNC 점수 표시
- 이전 브라우저 학습값이 새 로직을 덮어쓰지 않도록 V49 학습키로 분리

GitHub Pages 업로드
