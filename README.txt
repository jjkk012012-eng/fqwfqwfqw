공장용 STEP 견적 계산기 V15

핵심 변경:
- 구매품: 예상단가 자동 입력 후 공장이 구매단가 칸에서 수정
- 재료비: kg/개 × 수량 × 재질별 시세/kg × 할증% 기준
- CNC: kg/개 × 수량 × kg당 예상시간 × 시간당 단가 + 셋업 + 홀/탭 단가
- 판금/절곡: 재료비 + 셋업 + 기본/개 + 절곡횟수 × 절곡단가 + 홀/타공 × 홀단가
- 절곡 자동값: 판면 방향 개수가 아니라 긴 hard edge/플랜지 패치 기반 초기값
- 홀 자동값: compact 원형/타원 feature loop 기반 초기값

주의:
STEP mesh만으로 절곡/홀을 100% 확정할 수는 없습니다.
이 앱은 자동 초깃값을 채우고 공장이 바로 수정해 견적을 확정하는 내부 견적 계산기입니다.
